# Data Science Salary Dashboard

An interactive Excel dashboard analyzing global Data Science salaries across different job roles, countries, and employment types.

## Features

- Interactive slicers for:
  - Job Title
  - Country
  - Employment Type

- KPI Metrics:
  - Median Salary
  - Job Count
  - Top Job Platform

- Visualizations:
  - Salary comparison by role
  - Salary by employment type
  - Geographic salary distribution map

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Dashboard Design

## Key Insights

- Compare salaries across Data Science roles.
- Analyze salary trends by country.
- Explore compensation differences between full-time, part-time, contract, and internship positions.
